# Drop `best.pt` here

Put the SAM-HeLa checkpoint at:

```
models/checkpoints/sam_hela/best.pt
```

Keep the filename `best.pt`. The app finds it automatically on startup.

To point at a file somewhere else instead, use the app's
**I/O → Output settings…** dialog or set
`EYE_LABELLER_SAM_HELA_LOCAL_PATH=/full/path/to/best.pt`.
